# Quality Documentation Manager

Use this guidance-only skill for documentation control and compliance readiness. It does not run scripts or external automation.

## When to use

- You need consistent document lifecycle controls.
- You want stronger traceability across quality records.
- You need better readiness for internal or external audits.

## Core workflows

### 1) Document lifecycle governance

- Define document classes, ownership, approval levels, and review cadence.
- Standardize versioning, revision history, and effective-date practices.
- Archive obsolete documents with clear retention rules.

### 2) Change control

- Require impact assessment for every controlled-document change.
- Link change records to training, process updates, and risk effects.
- Track overdue change actions and unresolved approvals.

### 3) Traceability and compliance

- Map quality records to procedures and regulatory requirements.
- Maintain a document index with ownership and review status.
- Ensure required signatures and approvals are complete and auditable.

### 4) Audit readiness

- Run periodic completeness and consistency checks.
- Simulate retrieval drills for high-risk document sets.
- Maintain CAPA-linked documentation evidence and closure proof.

## Risk controls

- Avoid uncontrolled copies of critical procedures.
- Prevent review-date lapses for high-impact documents.
- Ensure access controls are role-appropriate and monitored.

## Output format

When asked for help, provide:

- A document control SOP structure.
- A change-control review checklist.
- An audit-readiness tracker template.
