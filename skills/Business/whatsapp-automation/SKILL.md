# WhatsApp Business Operations

Use this guidance-only skill to design WhatsApp customer communication processes. It does not execute external tool calls.

## When to use

- You need structured inbound and outbound communication policies.
- You want consistent message quality and faster response times.
- You need governance for templates and escalation paths.

## Core workflows

### 1) Template and messaging governance

- Define approved template categories and use cases.
- Standardize tone, personalization boundaries, and escalation language.
- Review template performance and retire low-performing variants.

### 2) Conversation operations

- Set response SLAs by message type and customer tier.
- Define handoff rules from bot/first-line support to specialists.
- Track unresolved conversations and overdue follow-ups.

### 3) Compliance and trust

- Define consent and opt-out handling policy.
- Limit sensitive data requests and use approved verification flows.
- Keep communication logs aligned with retention requirements.

## Risk controls

- Avoid template overuse that reduces customer relevance.
- Prevent SLA misses with threshold alerts and ownership.
- Escalate high-risk conversations quickly with clear accountability.

## Output format

When asked for help, provide:

- A message governance policy.
- A triage and escalation matrix.
- A communication KPI dashboard template.
