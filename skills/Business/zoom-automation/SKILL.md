# Zoom Meeting Operations

Use this guidance-only skill to improve Zoom collaboration and event workflows. It does not execute external tools.

## When to use

- You need consistent meeting scheduling and facilitation standards.
- You want clearer webinar planning and ownership.
- You need stronger post-meeting action follow-through.

## Core workflows

### 1) Meeting governance

- Define meeting types, durations, and required metadata.
- Require agenda, outcomes, and owner before scheduling.
- Protect focus time with no-meeting windows and rules.

### 2) Webinar and event operations

- Define role responsibilities for host, producer, and presenter.
- Use pre-event readiness checks (content, logistics, contingencies).
- Capture attendee signals and follow-up segmentation.

### 3) Post-meeting execution

- Record decisions, action items, owners, and due dates.
- Track action closure rates and unresolved carryovers.
- Run periodic quality reviews on recurring meetings.

## Risk controls

- Avoid recurring meetings without measurable outcomes.
- Prevent lost actions with explicit owner + deadline tracking.
- Escalate high-impact unresolved items in weekly operations review.

## Output format

When asked for help, provide:

- A meeting governance policy.
- A webinar readiness checklist.
- A post-meeting action tracker template.
