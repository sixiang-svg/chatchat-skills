# Shopify Operations

Use this guidance-only skill to design and optimize Shopify business operations. It does not run API calls or external tooling commands.

## When to use

- You need consistent product, inventory, and order workflows.
- You want better control over promotions and merchandising.
- You need operational visibility into fulfillment and returns.

## Core workflows

### 1) Catalog and merchandising governance

- Standardize product data requirements (title, media, specs, tags).
- Define collection logic and periodic cleanup rules.
- Set promotion guardrails for margin protection and campaign clarity.

### 2) Order and fulfillment operations

- Define exception paths for failed payments, splits, and delays.
- Track fulfillment SLA adherence and root causes of misses.
- Standardize customer communication templates for order issues.

### 3) Inventory and demand visibility

- Set low-stock and overstock thresholds by SKU class.
- Align replenishment cadences with campaign calendars.
- Review stockouts and dead-stock trends weekly.

### 4) Returns and post-purchase quality

- Define return reason taxonomy and policy enforcement.
- Track avoidable returns and feed insights into product pages.
- Publish post-purchase KPI reviews with owners and actions.

## Risk controls

- Avoid overlapping discounts that erode margin unexpectedly.
- Prevent stale product data from harming conversion quality.
- Keep high-risk order exceptions escalated with clear ownership.

## Output format

When asked for help, provide:

- A Shopify operations checklist.
- An order exception handling playbook.
- A weekly merchandising and fulfillment KPI template.
