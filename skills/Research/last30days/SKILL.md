# Last 30 Days

Research a topic from the last 30 days on Reddit + X + Web, become an expert, and write copy-paste-ready prompts for the user's target tool.

## Instruction
- Identify the target topic and search for discussions on Reddit, X, and the broader web from the last 30 days.
- Synthesize the most recent community signals, technical debates, and trending opinions into a cohesive expertise summary.
- Analyze shifts in consensus or new developments that occurred after the training cutoff of standard LLMs.
- Create high-fidelity, "copy-paste-ready" prompts tailored to specific AI tools based on the researched context.
- Highlight critical technical changes, emerging best practices, or security warnings mentioned in recent developer community feeds.
- Provide a clear roadmap for the user to implement the latest findings in their specific project or research.

## When to Use
- When researching cutting-edge topics that are evolving faster than standard LLM knowledge cutoffs (e.g., AI agents, new scientific breakthroughs).
- When needing an expert synthesis of community-driven discussions on platforms like Reddit and X.
- When developing modern prompts or content that must reflect the absolute latest industry trends.

## Output
- A synthesized briefing on recent technical trends and community consensus.
- Actionable, tool-specific prompts optimized for the most current information.
- Summary of goals, proposed next steps, and technical precautions for the target topic.
