# ETE Toolkit

Phylogenetic tree toolkit (ETE). Tree manipulation (Newick/NHX), evolutionary event detection, orthology/paralogy, NCBI taxonomy, visualization (PDF/SVG), for phylogenomics.

## Instruction
- Parse phylogenetic trees from standard Newick or NHX formats for manipulation and structural analysis.
- Conduct evolutionary event detection to identify speciation and gene duplication events within the tree.
- Analyze orthology and paralogy relationships by traversing tree nodes and comparing taxonomic markers.
- Annotate tree nodes with external metadata, such as NCBI taxonomy lineages or functional protein descriptions.
- Generate high-quality visualizations of trees and export them as publication-ready PDF or SVG files.
- Customize tree styles, including layout (circular/rectangular), branch labeling, and color-coded node highlighting.

## When to Use
- When performing phylogenomic analysis to determine evolutionary relationships between genes or species.
- When detecting gene family expansion/contraction or locating specific evolutionary nodes.
- When visualizing complex phylogenetic data for publication or research presentation.

## Output
- Analyzed tree structures with identified evolutionary events and annotated nodes.
- Orthology/paralogy mapping results for the provided phylogenetic dataset.
- High-resolution tree visualizations in PDF, SVG, or PNG formats.
