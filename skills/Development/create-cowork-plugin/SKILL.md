# Create Cowork Plugin

Guide plugin development from scope definition to implementation and operational readiness.

## When to Use

- You are implementing a new plugin for a cowork/productivity platform.
- You need structured guidance on integration, auth, and lifecycle events.

## Workflow

1. Map plugin capabilities to host extension points.
2. Define data contracts, auth flow, and permission boundaries.
3. Design user-facing actions, settings, and error states.
4. Implement lifecycle hooks, retries, and observability.
5. Provide packaging, versioning, and release checklist.

## Output

- Plugin architecture and integration plan
- Security and reliability checklist
- Release and maintenance guidance
