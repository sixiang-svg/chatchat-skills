# Algorithmic Art

Use this skill when building procedural or generative visual outputs through code.

## When to Use

- You want visual outputs driven by rules and randomness.
- You need reproducible generative patterns.
- You want adjustable parameters for style exploration.

## Workflow

1. Define style objective (geometry, color, density, motion).
2. Choose rendering stack and coordinate system.
3. Define parameter set (seed, iterations, palette, noise scale).
4. Implement generation loop with deterministic seed handling.
5. Add preview and parameter sweep for tuning.
6. Export final outputs at target resolution and format.

## Output

- Algorithm outline and parameter spec
- Reproducibility notes (seed and version)
- Export instructions (format, dimensions, naming)
