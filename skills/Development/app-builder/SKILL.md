# App Builder

Turn product goals into a concrete implementation plan and starter structure that developers can execute immediately.

## When to Use

- You have a PRD and need a development-ready app plan.
- You need feature breakdown, architecture, and implementation sequencing.

## Workflow

1. Extract user flows, entities, and acceptance criteria from the requirements.
2. Propose a modular architecture aligned to the requested stack.
3. Break work into milestones (MVP first, then enhancements).
4. Provide starter scaffolding guidance for backend, frontend, and shared contracts.
5. Call out delivery risks, dependencies, and unknowns.

## Output

- Architecture and module map
- Milestone-based implementation plan
- Suggested project scaffold and task checklist
