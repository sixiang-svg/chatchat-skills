# Game Design

Create actionable game design plans that can be implemented and tested by development teams.

## When to Use

- You are defining or revising gameplay systems.
- You need to translate design intent into implementation-ready specs.

## Workflow

1. Define core loop, player goals, and success metrics.
2. Specify mechanics, progression rules, and balancing levers.
3. Map UX flow for onboarding, challenge, and reward pacing.
4. Identify dependencies with art, audio, and engineering systems.
5. Provide playtest hypotheses and iteration checkpoints.

## Output

- Game system design document outline
- Balance and progression guidelines
- Playtest and iteration checklist
