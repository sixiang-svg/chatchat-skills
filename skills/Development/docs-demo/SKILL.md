# Docs Demo

Create practical docs that help users quickly understand and validate product capabilities.

## When to Use

- You need demo-quality docs for launches, onboarding, or PRs.
- Existing docs are too abstract and lack concrete examples.

## Workflow

1. Define audience goals and prerequisite knowledge.
2. Select one end-to-end scenario for demonstration.
3. Document setup, steps, outputs, and troubleshooting tips.
4. Add copy-paste examples and expected results.
5. Validate instructions against real execution path.

## Output

- Demo documentation draft
- Example snippets and outputs
- Review checklist for clarity and accuracy
