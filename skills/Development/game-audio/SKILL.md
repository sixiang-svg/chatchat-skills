# Game Audio

Build game-audio pipelines that support immersion, clarity, and performance constraints.

## When to Use

- You are implementing or improving game sound systems.
- You need structured guidance for event-driven audio and mixing.

## Workflow

1. Map gameplay events to required SFX/music cues.
2. Define audio asset specs, naming, and loudness targets.
3. Plan runtime triggers, spatialization, and mixing behavior.
4. Add adaptive audio states for game context transitions.
5. Validate latency, memory usage, and platform consistency.

## Output

- Audio system design and event map
- Mixing and integration rules
- Performance and QA checklist
