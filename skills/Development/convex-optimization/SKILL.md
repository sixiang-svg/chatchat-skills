# Convex Optimization

Translate optimization goals into solvable convex models and actionable implementation guidance.

## When to Use

- You need to optimize a system under constraints with mathematically reliable methods.
- You need help validating convexity and selecting a solver strategy.

## Workflow

1. Define objective, variables, and hard constraints precisely.
2. Verify convexity assumptions and feasible domain.
3. Select suitable solver/tooling and numerical strategy.
4. Interpret results and sensitivity to parameter changes.
5. Provide implementation-ready model specification.

## Output

- Problem formulation and convexity check
- Solver recommendation and setup notes
- Result interpretation and caveats
