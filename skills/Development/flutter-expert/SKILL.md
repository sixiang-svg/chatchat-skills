# Flutter Expert

Deliver maintainable Flutter solutions with clear structure, predictable state flow, and strong UX quality.

## When to Use

- You are implementing or refactoring Flutter features.
- You need guidance on architecture, performance, or platform-specific behavior.

## Workflow

1. Clarify feature behavior, platform needs, and state complexity.
2. Propose screen/widget architecture and state management approach.
3. Implement interaction flow, validation, and async error handling.
4. Optimize rendering, rebuild patterns, and resource usage.
5. Provide testing strategy for widget, integration, and device-level checks.

## Output

- Flutter implementation and architecture plan
- Performance and platform considerations
- Test and rollout checklist
