# Game Art

Guide game-art creation with consistent style, production efficiency, and engine-ready outputs.

## When to Use

- You are creating or scaling game visual asset production.
- You need to align art quality with runtime constraints.

## Workflow

1. Define visual style rules, palette, and asset taxonomy.
2. Specify asset specs (sizes, formats, naming, pivot/rig requirements).
3. Plan pipeline from concept to final engine import.
4. Add optimization guidance (atlas, LOD, compression).
5. Provide QA checks for style, readability, and performance.

## Output

- Game-art production plan
- Technical asset specification
- Integration and QA checklist
