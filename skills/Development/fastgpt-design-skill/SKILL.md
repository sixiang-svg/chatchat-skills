# FastGPT Design Skill

Create FastGPT skill designs that are actionable, testable, and production-ready.

## When to Use

- You are creating or refining FastGPT workflows/skills.
- You need structured prompt and tool orchestration guidance.

## Workflow

1. Define task boundaries, success criteria, and fallback expectations.
2. Specify prompt structure, variables, and tool invocation rules.
3. Design node flow for retrieval, reasoning, and action phases.
4. Add guardrails for unsafe or low-confidence outputs.
5. Create test prompts and evaluation checklist.

## Output

- FastGPT workflow/skill design
- Prompt and tool contract definitions
- Evaluation and iteration checklist
