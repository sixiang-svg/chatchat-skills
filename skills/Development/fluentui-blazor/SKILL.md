# Fluentui Blazor

Implement Fluent UI in Blazor apps with maintainable component composition and UX consistency.

## When to Use

- You are developing Blazor UIs with Fluent UI components.
- You need guidance for accessibility, variants, and component architecture.

## Workflow

1. Map UI requirements to Fluent UI component primitives.
2. Define component composition and state/binding strategy.
3. Apply consistent design tokens and interaction states.
4. Ensure keyboard, focus, and screen-reader accessibility.
5. Provide testing and visual regression checks.

## Output

- Fluent UI Blazor implementation plan
- Accessibility and consistency guidelines
- QA checklist
