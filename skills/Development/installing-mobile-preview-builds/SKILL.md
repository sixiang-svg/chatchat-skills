# Installing Mobile Preview Builds

Ensure preview builds reach testers quickly with minimal setup friction and reliable feedback loops.

## When to Use

- You are shipping preview builds to QA/internal testers.
- You need platform-specific install and troubleshooting guidance.

## Workflow

1. Validate build signing, versioning, and distribution readiness.
2. Select distribution channel and configure tester access.
3. Provide step-by-step install instructions per platform.
4. Add troubleshooting paths for common install failures.
5. Define feedback, crash reporting, and build expiry process.

## Output

- Preview build distribution plan
- Tester install guide
- Validation and feedback checklist
